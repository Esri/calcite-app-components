#### Basic

Renders a basic flow with a couple `calcite-flow-item`s.

```html
<calcite-flow>
  <calcite-flow-item heading="one, two, three, four">
    <!-- image -->
  </calcite-flow-item>
  <calcite-flow-item heading="tell me that you love me more">
    <!-- image -->
  </calcite-flow-item>
</calcite-flow>
```
